Abstract

This project implements a simplified version of the classic Pacman game, designed as a custom Reinforcement Learning (RL) environment using Python and the Turtle graphics library. The main objective is to train an intelligent Pacman agent to navigate a maze, avoid static ghosts, and collect rewards by visiting and exploring the environment.
